How to define a table?

"tr" stands for table rows

"th" stands for table header

"td" stands for table data

"thead" groupd the header content in the table

"tfoot" groups the footer content in the table

"caption" defines the table caption

What is the difference between colspan and rowspan?

Colspan makes a cell span over multiple column whereas the rowspan makes a cell psan over multiple rows

What is the difference between cell spacing and padding?

Cell padding is the space between the cell edges and the cell content. Cell spacing is the space between each cell.
